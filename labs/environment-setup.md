# Environment Setup

## Objective

To establish a structured lab environment for performing practical penetration testing exercises.

---

## Hardware Setup

- Primary Device: Windows 11 Laptop  
- Secondary Device: MacBook (backup system)  
- External SSD used as centralized lab storage  

---

## Storage Structure

```text
CyberLab/
├── ISOs/

```

## Operating Systems

- Kali Linux 2025.3 (Primary attack machine)  
- Windows 10 22H2 (Target system)  

---

## Lab Design

- Perform controlled attack simulations in a safe environment  
- Focus on basic penetration testing techniques  
- Document all steps, outputs, and observations  

---

## Workflow

1. Setup environment  
2. Execute attack  
3. Capture results (logs, screenshots)  
4. Document findings  
5. Store results in repository
